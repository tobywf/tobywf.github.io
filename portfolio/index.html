Caltech

Proof-of-concept web application

My Summer Undergraduate Research Fellowship was with the ground-breaking LIGO project, which in 2016 announced the discovery of gravitational waves.
This project manipulated high resolution astronomical data and rendered the resulting “skymaps” in a browser. Since everything had to happen during page loading, this meant processing a lot of data very quickly (< 100ms).
The successful proof-of-concept showed that a correctly engineered web application could deal with large amounts of data, and opened up the possibility of a more collaborative workflow compared to desktop applications.
